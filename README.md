# 🎓 Campus Connect

A modern **Lost & Found Management System** built using **React Native (Expo)** and **Firebase**, designed to help students and staff report, search, match, and recover lost belongings within a campus.

---

## 📱 Overview

Campus Connect simplifies the process of reporting and finding lost items by providing a centralized platform with real-time updates, intelligent search, secure authentication, and in-app communication.

---

## ✨ Features

- 🔐 Firebase Authentication (Email & Password)
- 📦 Report Lost Items
- 📍 Report Found Items
- 🔍 Advanced Search & Filters
- 🤝 Lost & Found Matching
- 💬 Real-Time Chat Between Users
- 📂 Post History
- ✅ Resolve/Archive Reports
- 👤 User Profile Management
- 📱 Responsive UI for Android

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/codin-priti/Campus_Connect.git
```

## Navigate to Project

```bash
cd Campus_Connect
```

## Install Dependencies

```bash
npm install
```

## Start Expo

```bash
npx expo start
```
